# 🛡️ Iridium Universal (v1.0-beta)

**Iridium** is a stateless, zero-dependency, cross-platform password generator. It uses cryptographic SHA-256 hashing to generate consistent, ultra-secure passwords without ever storing them.

Built to be truly universal, it runs natively on **Windows**, **Linux (WSL)**, and **Android (Termux)** without requiring any external Python libraries.

---

## 🔒 Security Architecture

Iridium follows the principle of **Separation of Concerns**:
* **The Code**: Public and open-source.
* **The Salt**: Private and stored only in your system's environment variables.

Your unique password is generated by: 
Master Password + Service Seed + System Environment Salt = Secure Key

## 🚀 Installation & Setup

### 1. Clone the repository
git clone https://github.com/danivsaubi/iridium.git
cd iridium

### 2. Set your Private Salt (CRITICAL)
The script requires an environment variable called IRIDIUM_SALT. 

**On Linux / WSL / Android (Termux):**
1. Edit your shell config: nano ~/.zshrc (or ~/.bashrc)
2. Add this line at the end:
   export IRIDIUM_SALT="your-very-secret-long-phrase-here"
3. Refresh your shell: source ~/.zshrc

**On Windows (PowerShell):**
[System.Environment]::SetEnvironmentVariable('IRIDIUM_SALT', 'your-secret-phrase', 'User')

### 3. Run the generator
python3 iridium.py

## ✨ Features
* **Double-Check**: Requires Master Password confirmation to prevent typos.
* **Auto-Cleanup**: Clears the terminal screen and wipes the system clipboard after 20 seconds.
* **Universal Clipboard**: Native support for clip.exe (Windows), termux-api (Android), and xclip/xsel (Linux).
* **Zero Dependencies**: Only uses standard Python libraries. No pip install required.

## 📱 Mobile Setup (Termux)
1. Install Termux and the Termux:API app from F-Droid.
2. Open Termux and run:
   pkg update && pkg install python termux-api git -y
3. Follow the Installation & Setup steps above.
4. (Optional) Add an alias for quick access:
   echo "alias iridium='python ~/iridium/iridium.py'" >> ~/.bashrc
   source ~/.bashrc
